## Características

- **Dark mode** con glassmorphism, gradientes y partículas animadas
- **Animaciones fluidas** con Framer Motion en cada sección
- **Cursor personalizado** con efecto glow
- **Typing effect** dinámico en el Hero
- **100% Responsive** — Mobile First
- **Filtros por tecnología** en Proyectos y Habilidades
- **Modal de detalle** por proyecto
- **Formulario de contacto** funcional
- **Scroll to top** animado
- **Loader** animado personalizado
- **Accesibilidad** — aria-labels en íconos interactivos

## Instalación Local

> **Solo para evaluación técnica. No para redistribución.**

```bash
# 1. Clonar el repositorio
git clone https://github.com/maxdev30/portafolio-dev.git
cd portafolio-dev

# 2. Instalar dependencias
npm install

# 3. Ejecutar en desarrollo
npm run dev

# 4. Build de producción
npm run build
```

## Personalización

Para actualizar los datos del portafolio, edita únicamente el archivo:

src/data/portfolio.js

Contiene toda la información: nombre, bio, skills, proyectos, experiencia y redes sociales.
